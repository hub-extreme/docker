Docker installation on ubuntu:
1. sudo apt-get update
2. sudo apt-get install curl
3. sudo curl -fsSl https://get.docker.com/ | sh
4. sudo usermod -aG docker ubuntu
5. sudo docker --version ........(checks  version of installed docker)  

  
Docker installation on amazon linux:
1. sudo yum update -y
2. sudo yum install docker -y
3. sudo systemctl status docker or sudo service docker status
4. sudo systemctl enable docker 
5. sudo systemctl start docker or sudo service docker start
6. sudo docker --version.
